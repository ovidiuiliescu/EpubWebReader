# Missing ARIA labels on icon-only buttons

## Severity
High

## Affected Files
- `src/components/Controls.vue` (lines 42-69, 76-101, 109-129, 133-191)

## Detailed Description
Multiple icon-only buttons in the Controls component use only the `title` attribute for accessibility. The `title` attribute is not sufficient for screen reader users as it is not announced on focus or when activated. These buttons lack `aria-label` attributes, making them inaccessible to keyboard-only and screen reader users.

Affected buttons include:
- Previous chapter navigation buttons (lines 42-52, 77-86)
- Next chapter navigation buttons (lines 58-68, 92-101)
- Decrease font size button (lines 109-117)
- Increase font size button (lines 121-129)
- Theme toggle button (lines 133-147)
- Wide mode toggle button (lines 150-158)
- Search button (lines 161-169)
- Table of contents button (lines 172-180)
- Library button (lines 183-191)

## WCAG 2.1 Criteria Violated
- **2.4.6 Headings and Labels**: Headings and labels describe topic or purpose
- **2.4.4 Link Purpose**: The purpose of each link can be determined from the link text alone or from the link text together with its programmatically determined link context
- **4.1.2 Name, Role, Value**: For all user interface components (including but not limited to: form elements, links and components generated by scripts), the name and role can be programmatically determined

## Implementation Plan

### Code Changes Required

1. **Add aria-label to navigation buttons** (lines 42-69, 76-101):
```vue
<button
  @click="bookStore.prevChapter()"
  :disabled="bookStore.currentChapter === 0"
  class="p-1.5 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800 disabled:opacity-50 disabled:cursor-not-allowed"
  :class="themeClasses.text"
  title="Previous chapter"
  aria-label="Previous chapter"
>
```

2. **Add aria-label to font size buttons** (lines 109-129):
```vue
<button
  @click="decreaseFontSize"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
  title="Decrease font size"
  aria-label="Decrease font size"
>
```

```vue
<button
  @click="increaseFontSize"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
  title="Increase font size"
  aria-label="Increase font size"
>
```

3. **Add aria-label to theme toggle** (lines 133-147):
```vue
<button
  @click="cycleTheme"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
  :title="`Current theme: ${themeLabel}`"
  :aria-label="`Change theme. Current theme: ${themeLabel}`"
>
```

4. **Add aria-label to wide mode toggle** (lines 150-158):
```vue
<button
  @click="settingsStore.toggleWideMode()"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800 hidden sm:block"
  :title="settingsStore.preferences.wideMode ? 'Wide screen mode on' : 'Wide screen mode off'"
  :aria-label="settingsStore.preferences.wideMode ? 'Wide screen mode on' : 'Wide screen mode off'"
>
```

5. **Add aria-label to utility buttons** (lines 161-191):
```vue
<!-- Search button -->

  title="Search"
  aria-label="Search in book"
>
```

```vue
<!-- TOC button -->
<button
  @click="emit('toggle-toc')"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
  title="Table of contents"
  aria-label="Table of contents"
>
```

```vue
<!-- Library button -->
<button
  @click="emit('toggle-library')"
  class="p-2 rounded-lg hover:bg-gray-100 dark:hover:bg-gray-800"
  title="Library"
  aria-label="Open library"

```
